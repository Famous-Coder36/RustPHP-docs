📦 Composer Installation

composer.json 
{
  "repositories": [
    {
      "type": "vcs",
      "url": "https://github.com/Famous-Coder36/RustPHP.git"
    }
  ],
  "require": {
    "famouscoder/rustphp": "*"
  }
}
Command 
composer require famouscoder/rustphp

🧪 Usage Example

<?php
require "vendor/autoload.php";

use RustPHP\Engine;

Engine::println("salom");

▶️ Run

php -d extension=./vendor/famouscoder/rustphp/target/debug/libssalom.so test.php

🛠 Build System

⚙️ PHP Requirements

  • PHP 8.2+
  • PHP CLI
  • php-dev headers
  • php-config binary

🐧 Install deps

# Debian
apt install php-dev

# Arch
pacman -S php

# Fedora
dnf install php-devel

🧱 Build PHP (advanced)

git clone https://github.com/php/php-src.git
cd php-src
git checkout PHP-8.1

./buildconf

PREFIX="$HOME/build/php"
./configure --prefix="$PREFIX" \
    --enable-debug \
    --disable-all --disable-cgi

make -j$(nproc)
make install
